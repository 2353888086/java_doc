/**
 * 多态的形式
 *     父类  对象名 = new 子类（）；
 *     接口  对象名 = new 实现类（）；
 *
 *     父类范围 大于 子类
 *
 * 多态是同一个行为具有多个不同表现形式或形态的能力，就是同一个接口，使用不同的实例而执行不同操作
 *
 * 同一个行为:
 *      意思是必输是父类方法，如果调用对象子类扩展的方法必须要强转为子类才能调用
 * 不同表现形式或形态的能力
 *      子类必须重写父类或实现接口方法，不然调用的就是父类的方法，就不存在不同的表现或形态能力一说
 *
 * 多态的识别
 *      方法：编译看左边执行看右边
 *      变量：编译看左边执行也看左边
 *          如果想调用子类变量必须强转为子类
 *
 * 多态的使用前提
 *      必须是继承或实现的关系
 *      必须是父类引用子类对象，或接口引用实现类对象
 *      需要存在方法重写或实现
 *
 */

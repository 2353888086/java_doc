/**
 * final修饰类：类不能被继承
 * final修饰方法：方法不能被重写
 * final修饰变量：这个变量必须是static且不能被修改且进被赋值一次
 *
 * 变量分类
 *      成员变量
 *             静态成员变量
 *                  static关键字修饰，随着类一起加载，且加载一次
 *             实例成员变量
 *                  无static修饰，随着对象一起加载，属于每个对象
 *      局部变量
 *             方法中的变量，方法结束变量就被释放
 *   final修改局部变量防止变量被修改
 *
 *   final修饰成员变量 只能赋值一次
 *      可以在代码块中赋值一次
 *      在每个构造器中赋值一次
 *      创建时赋值一次
 *
 */
